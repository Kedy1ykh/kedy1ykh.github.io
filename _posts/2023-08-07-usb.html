The 4 Protocols layer

- Transaction Packet(TP)

- Data Packet(DP)

- Isochronous Timestamp Packet(ITP)

- Link Management Packet(LMP)

All protocol start with 16 bytes header which can be split into

-12 bytes of header infomation (start with 5 bit of types [4:0])

-the 5 bits of type refer to the 4 varient of protocol layer above

-2 bytes of link control word

-2 bytes of crc(crc-16)

USB 4 Tunneling

HS0/1_TX/RX dedicated DP 1.2 lane

HS0/1_TX/RX dedicated PCIE 2.0 lane

USB3 lane

Control lane

-configure usb feature in usb router

-configure connection manager

each tunnelling has their own adaptor

each adaptor is define as followng type of connection

Adaptor: DN(source),UP(sink),IN(source),OUT(sink),Host interface & Control,CTL (config router)
